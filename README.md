# libipfs

Crystal bindings for go-libipfs

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     cr-libipfs:
       github: teknomunk/cr-libipfs
   ```

2. Run `shards install`

## Usage

```crystal
require "libipfs"
```
